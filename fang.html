🔗 连接 Facebook / Instagram
贴上 Page Access Token，自动抓最近的 FB 贴文和 IG 贴文填进来。默认不保存 Token；勾选下方「记住」则只存在这台设备的浏览器里。
所需权限：pages_show_list、pages_read_engagement、pages_read_user_content、read_insights、instagram_basic、instagram_manage_insights。
想要 60 天长效 Token：在 Graph API Explorer 生成 Token 后，到 Access Token Debugger 贴上 → Debug → 点底部「Extend Access Token」，把延长后的 Token 贴到下面并勾选「记住」。
注意：在 Claude 预览窗里抓取会被网络限制挡住——把本文件下载到电脑、用浏览器打开（或放到你自己的空间）就能正常用。曝光用的是 2026 年 6 月起的新指标（views / post_media_view），少数贴文若拿不到会显示 0，可再用截图补。
