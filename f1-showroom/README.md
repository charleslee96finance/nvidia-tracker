# F1 暗夜车库实景 3D

法拉利 F1-75（蒙扎特别涂装）× 红牛 RB21 互动 3D 展厅，**双场景一键切换**：

- 🏜️ **摩洛哥赛道**（默认）—— 马拉喀什街道赛风格：发车格与黑白起步线、
  红白路肩、起步灯架、赭红城墙与马蹄拱门、棕榈树、摩洛哥国旗、观众看台、
  阿特拉斯雪山背景。可切换 **☀️ 正午 / 🌅 黄昏 / 🌙 夜战** 三种时段
  （黄昏金色斜阳、夜战泛光灯架 + 星空，光照与天空整体联动）
- 🌃 **暗夜车库** —— 对照实拍照片还原：链条吊挂灯管、聚光光池、
  高反光水泥地、门洞体积光

车身涂装对照实拍照片，1:1 实车比例；全部车体与场景为 three.js 程序化建模，
**零外部资源**，无需联网。

## 功能

- **360° 自由旋转**：拖动环绕、滚轮/双指缩放、右键平移，支持自动旋转展示
- **驾驶舱第一视角**：坐进座舱，眼前是 halo 保护圈与方向盘仪表屏
  （动态转速灯条 / 档位 / 车速 / ERS 电量 / DRS / 圈数实时模拟）
- **分层结构**：
  - 完整车身
  - 透视外壳（X 光半透明）
  - 拆解模式 —— 引擎盖上掀、侧箱滑出、鼻锥前移，露出
    V6 涡轮混动引擎、涡轮增压器、排气歧管、变速箱、散热器、
    ERS 电池组、MGU-K 等内部细节（带中文标注）
- 点击任意一辆赛车即可聚焦；地面指示牌与右下角参数卡对应当前车辆
- **🚦 起跑演示**（赛道场景）：五盏红灯依次亮起 → 随机熄灭 → 两车轮胎转动、
  并驾加速冲出直道，镜头跟拍；途中 DRS 尾翼自动开启。结束自动复位发车格
- **🔊 引擎声浪**：Web Audio 实时合成 V6 涡轮声，音高/音量跟随仪表盘转速与
  油门变化（起跑时随升档轰鸣），可一键开关
- **细节动画**：DRS 襟翼随 DRS 区开合、排气口热浪扰动、起跑时车轮转动 + 前轮微摆、
  后轮起步扬沙尾流
- **📷 拍照模式**：一键隐藏全部界面，留下干净画面；可保存当前视角为 PNG 壁纸
  （Esc 退出）
- **中 / 英双语**：面板 🌐 按钮一键切换（界面、参数卡、引擎标注全部翻译，
  选择会记住；URL 加 `#lang=en` 可直接以英文打开）

## 使用

直接用浏览器打开 `index.html` 即可（纯静态、非模块脚本，支持 file:// 协议）。

也可以起一个本地服务：

```bash
cd f1-showroom
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## 文件

| 文件 | 说明 |
| --- | --- |
| `index.html` | 页面外壳与控制面板 UI |
| `app.js` | 全部 3D 逻辑：车库场景、程序化造车、相机控制、拆解动画、仪表模拟 |
| `three.min.js` | three.js r128（与仓库内太阳系项目同版本） |
